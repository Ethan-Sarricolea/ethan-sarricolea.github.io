    Proyectos

    • Generador de mazmorras aleatorias

      Proyecto escolar que genera mazmorras (representadas en grafos) con un numero de salas repartidas y conectadas de forma aleatoria, este proyecto se desarrollo con un pequeño juego para recorrer las salas ganando o perdiendo puntos de acorde a la cantidad de salas recorridas y, finalmente al terminar la sala saber que camino se recorrio, cuantos puntos se obtuvieron o perdieron al final y cual era el camino mas corto.

    • Organizador de archivos

      Aunque es un proyecto simple y sencillo, implico en su momento un paso mas en la escala de mis conocimientos adquiridos. este proyecto hecho con dart permite organizar en carpetas diferentes archivos dentro de una ruta seleccionada. Ademas de ello generé una pequeña aplicación ejecutable con python para realizar la organización rapidamente.

    • Modelo de regresión lineal
